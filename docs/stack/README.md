【题目四】 一个栈依次压入1、2、3、4、5,那么从栈顶到栈底分别为5、4、3、2、1。将这个栈 转置后,从栈顶到栈底为1、2、3、4、5,也就是实现栈中元素的逆序,但是只能用 递归函数来实现,不能用其他数据结构。
> 递归其实相当于一个栈结构。
